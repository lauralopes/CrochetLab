# 🧪 CrochetLab

Welcome to **CrochetLab**, a creative crochet lab where every pattern is crafted with care, yarn, and personality.  
Here you'll find original tutorials for accessories, home decor, and unique pieces — all tested and documented with photos and step-by-step instructions.

---

## 📂 Pattern Collections

- `patterns/` — Crochet patterns written in English
- `patterns-pt/` — Receitas de crochê escritas em português

---

## 🧵 Contributions

Feel free to suggest improvements, share your own versions, or submit new patterns via pull requests!

---

## 📸 Gallery

Project photos are stored in the `images/` folder and linked inside each pattern.
